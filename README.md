# java-racingcar
우아한테크코스 레벨1 자동차 경주 게임

**N대의 자동차가 M번만큼 경주를 하여 우승자를 구하는 게임**

## 기능 목록
1. 자동차 객체 설정
    - 이동 전략에 따라 전진 혹은 정지
    - 이름과 현재 위치를 가짐
2. 자동차 이름, 시도 횟수 입력받기
    - 자동차 이름은 쉼표를 기준으로 구분
    - 자동차 이름은 5자 이하로 구성
    - 자동차 이름은 2개 이상으로 한다
    - 자동차 이름은 중복되지 않는다
    - 공백, 쉼표, 오타 보정
3. 이동 전략 설정
    - 0 ~ 9 범위의 랜덤 넘버를 생성 후 4 이상일 경우 전진, 3 이하일 경우 정지
    - 테스트용 무조건 전진 전략 설정
4. 시도 횟수 입력
    - 1 이상의 정수
5. 경주 실행 결과 표시
    - 매 시도마다 모든 자동차의 포지션을 표시
6. 우승자 판별 및 표시
